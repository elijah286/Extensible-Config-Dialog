Unit Testing

Configure which LabVIEW unit-test frameworks run in CI and where their tests live. Results publish to the dashboard as a single Unit Tests report per revision, with links straight to the VI under test in the VI Browser.

1. Repository

These settings apply to the repository you came from — its current configuration is loaded automatically.

Configuring … on branch main

2. Frameworks

Enable a framework and tell it where its tests live. Several can run at once — results merge into one report. Written to .github/labview-ci.yml under unitTests.

3. Badge thresholds

The dashboard badge shows the percentage of tests that failed. Colours use pass-rate thresholds, which is the usual way to define test-health gates.

4. Review & apply

Save & Apply queues the Reconfigure workflow with the dashboard's stored token. The workflow updates only the unitTests block and opens a pull request to review. Download/copy options remain available for manual changes.
