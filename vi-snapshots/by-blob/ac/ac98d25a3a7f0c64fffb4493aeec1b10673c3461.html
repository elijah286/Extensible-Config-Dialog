Configuration.lvlib:Configuration.lvclass:Actor Core.vi

(Filename: Actor Framework.lvlib:Actor.lvclass:Actor Core.vi)

Receives messages to the actor, reacts to them, and initiates any error handling that may shut the actor down.

Descendant classes may override this VI to append parallel tasks for the actor to complete while it handles messages. Overrides use the Call Parent Method node.

Controls and Indicators

 error in

error in describes error conditions that occur before this node runs. This input provides standard error in functionality.

    		status

    status is TRUE (X) if an error occurred before this node ran or FALSE (checkmark) to indicate a warning or that no error occurred before this node ran. The default is FALSE.

    		code

    code is the error or warning code. The default is 0.

    		source

    source specifies the origin of the error or warning and is, in most cases, the name of the node that produced the error or warning. The default is an empty string.

Configuration in

Actor in specifies the state of the actor after it has launched.
